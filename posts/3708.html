第二章 对国家出路的早期探索

太平天国运动

金田起义

永安建制

1853年 –> 天朝田亩制度 1859 –> 资政新篇

1856年 –> 天京事变 –> 太平天国由盛转衰的标志

1864年 –> 太平天国唧唧时间

太平天国的历史意义：

  1. 沉重打击封建统治阶级，撼动了清政府的统治根基。
  2. 批判儒家为代表的封建统治思想。
  3. 打击外国势力，维护中国主权。
  4. 掀起亚洲反殖民主义的大潮。
  5. 两个文件让中国旧式农民起义达到最高峰。

天平天国的失败原因：

  1. 缺乏先进的阶级领导。（根本原因）
  2. 没有科学理论的指导。
  3. 笼统地把信奉上帝的西方人称为“洋兄弟”。

洋务运动

目的 –> 为有效镇压太平天国起义，从近代军用工业着手。

首领 –> 奕䜣

指导思想：中学为体，西学为用。 –> 冯桂芬

办事机构：1961年 –> 总理各国事务衙门

较大的洋务企业：江南制造总局

新式海军 –> 北洋水师

近代中国第一个翻译学堂 –> 京师同文馆

洋务运动的历史意义：

  1. 工业：促进工业和民族资本主义的发展。
  2. 教育：成为近代教育的开端。
  3. 文化：传播新知识，打开了人们的眼界。
  4. 社会：引起社会风气和价值观念的变化。

洋务运动的失败原因：

  1. 洋务运动具有封建性。
  2. 洋务运动对西方列强具有依赖性。
  3. 洋务企业的管理具有腐朽性。
  4. 顽固派的阻挠。

戊戌变法（维新变法、百日维新、戊戌维新）

康有为 –> 三考一理

梁启超 –> 变法通议 时务报

谭嗣同 –> 仁学

维新派和守旧派的论战 –> 资产阶级思想和封建主义思想在中国的第一次正面交锋。

张之洞在劝学篇引用 –> 中学为体，西学为用 <– 对抗维新变法

戊戌变法的历史意义：思改国资

  1. 是一场思想启蒙运动。
  2. 在改革社会风气方面也有不可低估的意义。
  3. 是一次爱国救亡运动。
  4. 是一场资产阶级性质的政治改革运动。

戊戌变法的失败原因：

  1. 慈禧太后为首的强大的守旧势力的反对。（主要原因）
  2. 不敢否定封建主义。
  3. 对帝国主义抱有幻想。
  4. 脱离人民群众。

辛亥革命的意义：

  1. 思：推动了思想解放。
  2. 变：促使社会经济、思想习惯和社会风俗发生了新变化。
  3. 清：推翻了清王朝在中国的统治。
  4. 帝：打击了帝国主义在华势力。
  5. 国：宣告了民国的建立。

辛亥革命的局限性：

  1. 党：没有建立坚强的革命政党。
  2. 人：没有充分的发动和依靠民众。
  3. 纲：没有提出彻底的反对帝国主义和封建主义的革命纲领。
