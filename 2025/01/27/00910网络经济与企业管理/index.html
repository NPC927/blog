自考00910网络经济与企业管理

第1章 企业管理概述

1.1 企业及其形式

（选择、文字）⭐⭐⭐⭐⭐企业的概念：1.企业是以市场为导向从事商品生产和流通的经济组织。2.企业是以价值增值作为其经济活动的目的。3.企业进行自主经营、自负盈亏和独立核算。

（单选）⭐⭐⭐⭐⭐企业发展的三个时期：手工业生产时期→工厂生产时期→现代企业生产时期。工厂制度的建立，标志着企业的真正形成。

（多选）⭐⭐⭐⭐⭐现代企业生产时期的特征：1.拥有现代技术。2.拥有现代化管理。3.所有者和经营者相分离。4.生产规模空前扩大，产生了垄断组织。5.劳动分工越来越精细，协作关系复杂、严密。6.企业之间的竞争激烈。7.企业的社会责任改变。

（选择）⭐⭐⭐⭐⭐企业的类型：

  • 根据企业所属的经济部门：农业企业、工业企业、金融企业、商业企业。

  • 根据生产力要素比重：劳动密集型企业、技术密集型企业、知识密集型企业、资源密集型企业。

  • 根据企业经营规模：大型企业、中型企业、小型企业。

  • 根据企业财产构成和所负法律责任：

    • 个体企业：又叫独资企业，是由业主个人出资兴办，由业主自己经营。在国外，许多零售商业、手工业、家庭工业、农业等多采用个体企业。
    • 合伙企业：由两个或两个以上的个人共同出资，通过签订协议联合经营。
    • 公司制企业：由两个以上的出资者共同出资、依法组建，以盈利为目的的企业，法人联合经营的企业组织形式。

（简答）⭐⭐⭐公司制企业的特点：1.公司制企业是法人。2.公司制企业实行有限责任制度。3.公司制企业的所有权和经营权相分离。

1.2 企业管理概述

（选择）⭐⭐⭐管理的概念：1.管理是一个动态过程。2.管理的目的是达到组织的预定目标。3.管理的载体是组织。4.管理的核心是对资源的合理配置和有效整合。

（多选）⭐⭐⭐管理活动需具备的条件：1.管理的主题（谁来管）。2.管理的客体（管理谁）。3.管理的目的（为啥管）。

（选择）⭐⭐⭐⭐法约尔”五职能论“：企业的管理活动就是由计划、组织、指挥、协调、控制这五种职能组成的。

（单选）⭐⭐⭐⭐⭐管理的计划职能：计划（首要职能）、组织、领导、控制是各管理学派普通公认的职能。计划职能是对既定的目标进行具体安排，作为全体员工在一定时期内的行动纲领，并规定实现目标的途径、方法的管理活动。（中期计划：1~5年）

（选择、简答）⭐⭐⭐⭐⭐组织工作的基本原则：有效性原则、统一指挥原则、责权利相一致原则、集权和分权相结合原则、弹性原则、协调原则。

（选择、简答）⭐⭐⭐领导的活动内容：1.权力的形成与运用。2.指导。3.激励。4.沟通。5.协调。6.营造组织气氛，建立组织文化。

（选择）⭐⭐⭐⭐⭐控制的方式：预先控制、现场控制（最基本的控制方式）、反馈控制。

（单选）⭐⭐⭐⭐企业管理的目标：1.企业是以价值增值作为其经济活动的目的。2.企业管理的目标是实现企业价值的最大化。

1.4 网络时代的企业环境

（文字、选择）⭐⭐⭐⭐网络时代企业面对的机遇

  1. 企业可以更好的满足顾客的个性化需求。
  2. 企业可以降低交易成本。
  3. 企业可以减少库存。
  4. 企业可以使合作竞争战略更便利地实施。
  5. 提高获取知识、应用知识的能力。

（文字、选择）⭐⭐⭐⭐网络时代企业面对的挑战

  1. 企业面临日益激烈的竞争。
  2. 顾客的权力大大增加。
  3. 企业的整体盈利水平将会降低。
  4. 企业关键的成功因素将会改变。
  5. 企业资源配置的方式和界限将会发生重大变化。

1.5 网络时代企业管理的变革

（选择）⭐⭐企业管理拓展的范围有哪些？

  1. 网络企业管理。
  2. 企业的网络化管理。
  3. 更加重视以知识资本为核心的无形资本的管理。
  4. 企业管理的范围拓展到整个供应链。

（选择）⭐⭐企业管理创新的内容有哪些？

  1. 企业战略管理的创新。
  2. 企业组织管理的创新。
  3. 网络营销成为营销管理的重要内容。
  4. 敏捷制造成为企业生产运作管理最重要的方式。
  5. 企业财务管理将向战略型、集成化方向发展。
  6. 回归人本管理，重视人力资源管理。
  7. 知识管理成为企业管理的重要内容。
  8. 更加重视文化管理。

第2章 企业战略管理

2.1 战略的性质与类型

（选择）⭐⭐⭐⭐⭐企业战略的三个层次。

  1. 企业总体战略：又称经营战略，是对企业整体经营的方向、原则、方针所作的规定，是企业如何运行的指导思想。
  2. 事业层战略：又称竞争战略，业务层次战略或者SBU战略，是指导和管理具体战略经营单位的计划和行动。
  3. 职能战略：又称职能部门战略，是企业研发、生产、营销、财务和人力管理等部门的短期战略计划。

（选择）⭐⭐⭐企业战略管理的特点（整体性、长期性、权威性、环境适应性）

（单选、简答）⭐⭐⭐⭐企业战略管理的过程

  1. 确定企业的使命。
  2. 明确企业的目的和目标。
  3. 企业战略条件分析。
  4. 制定战略方案。
  5. 战略方案选优。
  6. 战略的实施与修正。

2.2 企业战略环境分析

（选择、简答）⭐⭐⭐企业环境的构成

  • 微观环境：包括那些直接影响企业履行其使命状况的行动者、供应商、各种市场中间商、顾客、竞争对手等。
  • 宏观环境：包括人口环境，经济环境，自然环境，技术环境，政治和法律环境，社会文化环境。宏观环境的三种状态：相对稳定状态，缓慢发展状态，动荡不定状态。

（多选、文字）⭐⭐⭐产业环境分析的内容：1.产业的特性。2.产业的寿命周期分析。3.产业的发展状况。4.产业的竞争结构分析。

（选择）⭐⭐⭐产业的生命周期（初创期、成长期、成熟期、衰退期）

（单选、简答）⭐⭐⭐⭐迈克尔·波特的五力竞争说（行业内竞争者、代用品、潜在的加入者、购买者、供应者）

2.3 企业资源和能力分析

（单选、文字）⭐⭐⭐⭐什么是企业的核心能力？核心能力是企业独有的、能够支撑企业长期竞争优势的能力。核心能力的要素：员工的知识和技能、技术体系、管理体系、价值观念与行为规范。

（选择、简答）⭐⭐⭐⭐⭐核心能力的特性：有价值的创造能力、异质性、难模仿性、动态性、扩展性、不可交易性。

（单选）⭐⭐⭐核心能力的载体（核心产品）

（单选）⭐⭐⭐核心能力的来源（企业所具有的资源、内聚能力）

（选择）⭐⭐⭐如何判断企业资源的价值？（稀缺性、持久性、可替代性）
